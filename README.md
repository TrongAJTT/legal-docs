# ⚖️ Legal Documents & Policies

Welcome to the central repository for all legal documents, privacy policies, and terms of service related to the software products, browser extensions, and web applications developed by Minh (TrongAJTT).

## 🎯 Why is this repository public?
**Transparency and Trust.** We believe that users have the right to know exactly how their data is handled and what terms they are agreeing to. By hosting these documents on a public GitHub repository, we ensure:
1. **Single Source of Truth:** All applications point to this unified repository for their latest legal terms.
2. **Version Control:** You don't have to take our word for what changed. Thanks to Git, you can view the complete, timestamped [Commit History](../../commits/main) of every document to see exactly what was modified, added, or removed over time.

## 🕒 How to view past versions
We do not maintain separate web pages for outdated policies. If you wish to view a previous version of any document:
1. Navigate to the specific document (e.g., `imify/terms.md`).
2. Click the **"History"** button (or the clock icon) in the top right corner of the file view on GitHub.
3. Browse the timestamped commits to see the exact state of the document on any given date.
